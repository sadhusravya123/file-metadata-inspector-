MetaView is a web application that extracts and displays metadata from uploaded Images and PDFs using the Browser File API, EXIF.js, and PDF.js.

Features
- Image & PDF Upload
- File Name, Size & MIME Type
- Image Dimensions
- EXIF Metadata (Camera, Date, GPS)
- PDF Metadata
- Drag & Drop Upload
- Dark Mode
- Copy Metadata
- Download Metadata as JSON
- Responsive UI

 Technologies
- HTML5
- CSS3
- JavaScript
- EXIF.js
- PDF.js

Run Locally
1. Clone the repository.
2. Open the project folder.
3. Run `index.html` using Live Server.
